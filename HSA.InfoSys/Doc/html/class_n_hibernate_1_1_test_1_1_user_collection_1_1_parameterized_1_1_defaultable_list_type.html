object NHibernate.Test.UserCollection.Parameterized.DefaultableListType.Instantiate ( int  anticipatedSize)

Instantiate an empty instance of the "underlying" collection (not a wrapper), but with the given anticipated size (i.e. accounting for initial size and perhaps load factor).

Parameters
anticipatedSizeThe anticipated size of the instantiated collection after we are done populating it. Note, may be negative to indicate that we not yet know anything about the anticipated size (i.e., when initializing from a result set row by row).

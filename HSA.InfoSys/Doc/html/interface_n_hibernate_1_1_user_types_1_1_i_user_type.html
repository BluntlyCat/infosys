Detailed Description

The interface to be implemented by user-defined types.

The interface abstracts user code from future changes to the Type.IType interface, simplifies the implementation of custom types and hides certain "internal interfaces from user code.

Implementers must be immutable and must declare a public default constructor.

The actual class mapped by a IUserType may be just about anything. However, if it is to be cacheble by a persistent cache, it must be serializable.

Alternatively, custom types could implement Type.IType directly or extend one of the abstract classes in NHibernate.Type. This approach risks future incompatible changes to classes or interfaces in the package.

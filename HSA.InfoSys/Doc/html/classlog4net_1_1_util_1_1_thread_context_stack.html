void log4net.Util.ThreadContextStack.Clear ( )

Clears all the contextual information held in this stack.

Clears all the contextual information held in this stack. Only call this if you think that this tread is being reused after a previous call execution which may not have completed correctly. You do not need to use this method if you always guarantee to call the IDisposable.Dispose method of the IDisposable returned from Push even in exceptional circumstances, for example by using the using(log4net.ThreadContext.Stacks["NDC"].Push("Stack_Message")) syntax.

IDisposable log4net.Util.ThreadContextStack.Push ( string  message)

Pushes a new context message into this stack.

Parameters
messageThe new context message.
Returns
An IDisposable that can be used to clean up the context stack.

Pushes a new context onto this stack. An IDisposable is returned that can be used to clean up this stack. This can be easily combined with the using keyword to scope the context.

Simple example of using the Push method with the using keyword.

using(log4net.ThreadContext.Stacks["NDC"].Push("Stack_Message"))
{
log.Warn("This should have an ThreadContext Stack message");
}

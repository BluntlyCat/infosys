Detailed Description

Support for Query By Example.

List results = session.CreateCriteria(typeof(Parent)) .Add( Example.Create(parent).IgnoreCase() ) .CreateCriteria("child") .Add( Example.Create( parent.Child ) ) .List();

"Examples" may be mixed and matched with "Expressions" in the same ICriteria

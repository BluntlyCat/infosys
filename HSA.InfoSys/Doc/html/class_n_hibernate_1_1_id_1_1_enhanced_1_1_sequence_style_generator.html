Detailed Description

Generates identifier values based on an sequence-style database structure. Variations range from actually using a sequence to using a table to mimic a sequence. These variations are encapsulated by the IDatabaseStructure interface internally.

General configuration parameters:

NAME DEFAULT DESCRIPTION
SequenceParam DefaultSequenceName The name of the sequence/table to use to store/retrieve values
InitialParam DefaultInitialValue The initial value to be stored for the given segment; the effect in terms of storage varies based on Optimizer and DatabaseStructure
IncrementParam DefaultIncrementSize The increment size for the underlying segment; the effect in terms of storage varies based on Optimizer and DatabaseStructure
OptimizerParam depends on defined increment size Allows explicit definition of which optimization strategy to use
ForceTableParam false Allows explicit definition of which optimization strategy to use

Configuration parameters used specifically when the underlying structure is a table:

NAME DEFAULT DESCRIPTION
ValueColumnParam DefaultValueColumnName The name of column which holds the sequence value for the given segment

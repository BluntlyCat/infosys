Detailed Description

Defines a contract for implementations that know how to convert a System.Data.Common.DbException into NHibernate's ADOException hierarchy.

Inspired by Spring's SQLExceptionTranslator.

Implementations must have a constructor which takes a IViolatedConstraintNameExtracter parameter.

Implementations may implement IConfigurable if they need to perform configuration steps prior to first use.

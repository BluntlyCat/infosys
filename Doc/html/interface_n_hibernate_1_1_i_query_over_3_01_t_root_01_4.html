Detailed Description

QueryOver<TRoot> is an API for retrieving entities by composing Criterion.Expression objects expressed using Lambda expression syntax.

IList&lt;Cat&gt; cats = session.QueryOver&lt;Cat&gt;()
.Where( c =&gt; c.Name == "Tigger" )
.And( c =&gt; c.Weight > minWeight ) )
.List();

Detailed Description

This is the base class that adds information to the DbType for the Driver.IDriver and Dialect.Dialect to use.

The Driver.IDriver uses the SqlType to get enough information to create an IDbDataParameter.

The Dialect.Dialect use the SqlType to convert the DbType to the appropriate sql type for SchemaExport.

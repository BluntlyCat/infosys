Detailed Description

Concrete implementation of an NHibernate.ISession, also the central, organizing component of NHibernate's internal implementation.

Exposes two interfaces: NHibernate.ISession itself, to the application and ISessionImplementor to other components of NHibernate. This is where the hard stuff is... This class is NOT THREADSAFE.

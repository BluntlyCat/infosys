Detailed Description

Base class to create queries in "detached mode" where the NHibernate session is not available.

See Also
IDetachedQuery, NHibernate.Impl.AbstractQueryImpl

The behaviour of each method is basically the same of NHibernate.Impl.AbstractQueryImpl methods. The main difference is on SetProperties(object): If you mix object with named parameters setter, if same param name are found, the value of the parameter setter override the value read from the POCO.

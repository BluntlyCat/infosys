Detailed Description

An unordered, unkeyed collection that can contain the same element multiple times. The .NET collections API, has no Bag. Most developers seem to use IList{T} to represent bag semantics, so NHibernate follows this practice.

Template Parameters
TThe type of the element the bag should hold.

The underlying collection used is an List{T}

Detailed Description

Optimizer which uses a pool of values, storing the next low value of the range in the database.

Note that this optimizer works essentially the same as the HiLoOptimizer, except that here the bucket ranges are actually encoded into the database structures.

Note that if you prefer that the database value be interpreted as the bottom end of our current range, then use the PooledLoOptimizer strategy.
